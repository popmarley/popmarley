# Hüseyin Özgüven

Merhaba, ben Hüseyin. GitHub'da **popmarley** kullanıcı adıyla; WordPress/WooCommerce eklentileri, Windows masaüstü araçları, otomasyon sistemleri ve günlük işleri kolaylaştıran küçük ama kullanışlı yazılım projeleri geliştiriyorum.

Odak noktam; **sade çalışan, gerçek bir ihtiyacı çözen, kullanıcı tarafında karmaşa oluşturmayan ve geliştirilebilir yapıda kalan yazılımlar** üretmek.

---

## 🚀 Odaklandığım Alanlar

* WordPress ve WooCommerce eklenti geliştirme
* Windows Forms / .NET tabanlı masaüstü uygulamaları
* Web arayüzleri ve yönetim panelleri
* Dosya dönüştürme, hesaplama ve üretkenlik araçları
* Excel entegrasyonlu otomasyon sistemleri
* Kullanıcı deneyimi odaklı pratik yardımcı uygulamalar
* Oyun yerelleştirme ve teknik çeviri araçları

* Önce ihtiyacı netleştirir, sonra en sade uygulanabilir çözümü üretirim.
* Gereksiz özellik kalabalığı yerine kullanılabilirlik ve bakım kolaylığına odaklanırım.
